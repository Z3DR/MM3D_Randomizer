# Version 1.4.4

## Features

## Fixes
- The fill algorithms have been updated in the Tokensanity logic. This should prevent freezes and failing to place items in the generator.
- Rename "Version" to "Game Version" in Game Settings Menu for clarity on which version it's referencing,
- Adjusts all vanilla Gold Skulltula Token locations to be non-repeatable locations by default,
- Added "Repeat Items on Tokens" toggle inside of Item Pool Settings menu which allows repeatable items (items potentially needed more than once in a play through such as Title Deeds or Anju & Kafei quest items) to be placed inside spider houses in vanilla gold skulltula token locations, in the case a repeatable item ends up on a skulltula it will be able to be gotten on new cycles,
- Updates settings so that if you start with Song of Healing you are unable to exclude the Deku Mask and Bombers Notebook locations in the Exclude Loactions menu preventing a generation issue.
- Adjusts Item Table to make less items "Repeatable"
- Adds variable to pass Randomizer Version and Commit ID to patch for use in Ingame Spoiler Log.
- Adjusts Spoiler Group for Inside Clock Tower checks so they have their own page.
- Adjusts Option text for `Console/Emulator` setting to `Console/Emulator` instead of `3DS/Citra` for clarity
- Adds some additional information to the in-game tracker, such as the commit ID and version number for easier bug tracking.
- Adds Randomizer Version and Commit ID to Seed Hash page for easier troubleshooting.
- Adds a new column to the Dungeon Items page to include stray fairies collected per dungeon.
- Adds two new rows below Dungeon Items to display the amount of gold skulltulas collected.
- Added new Inside Clock Tower page to contain checks from the Happy Mask Salesman.
- Fix Koume not giving Pictograph Box if attempting to obtain after defeating Odolwa and going to a new cycle.
