# Release Notes

This is an archive of all the changes from each nightly and full version release, starting at [V1.4.3](https://github.com/Z3DR/MM3D_Randomizer/releases/tag/v1.4.3) and [Nightly E0EE25](https://github.com/Z3DR/MM3D_Randomizer/releases/tag/Nightly-e0ee25).

To view the latest release changes, please click on [Latest Stable Release Changes](./changelog-latest-release.md).  
To view the last Nightly changes, please click on [Latest Nightly Changes](./changelog-latest-nightly.md).

**Please note** that nightlies will tend to have a few outstanding bugs while new features are implemented. There is always a chance that you can softlock your run, or the randomizer may be prone to crashing. While full version releases may have some bugs, they are usually not enough to completely ruin a playthrough. If you're ever stuck, please reach out to us over [Discord](https://discord.com/invite/73ZNdjbfFp) and we should be able to assist you!
