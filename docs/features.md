# Features
Here is a comprehensive list of what is available in the Randomizer if you are unable to launch the generator.

## Randomized Items

- Main Inventory Items (Anything on the Gear Screen)
- Transformation Masks (Deku, Goron, Zora & Fierce Deity)
- Masks
- Great Fairy Rewards
- Small Keys
- Big Keys
- Dungeon Maps & Compasses
- Heart Containers
- Boss Remains
- Tingle Maps
- Pieces of Heart
- Anju & Kafei Quest Items
- Deku Merchant Trades (Title Deed Questline)
- Bomber's Notebook
- Skulltula Tokens (Tokensanity)
- Songsanity + Song of Time & Song of Soaring separate options
- Ocarina Shuffle
- Moon Items

## Options

- Ice Trap Settings (Quantity only not different types, yet)
- Starting Inventory (Including starting without Sword/Shield and Ocarina)
- Ingame Spoiler Log
- Gossip Stones Hint System
- Chest Size Matches Content
- Blast Mask Cooldown
- Exclude Locations
- Remove Double Defense
- Custom Button Mapping (option to assign Touch Screen buttons to physical buttons & change Notebook and Ingame Spoilers buttons)
- Item Pool difficulty settings
- Transformation Mask Custscene Skips (Shortens but does not entirely skip Deku, Goron, and Zora cutscenes)
- Fierce Deity Anywhere
- Damage Multiplier
- Skip longer cutscenes (i.e. Giant's Cutscene on Rooftop, and Zora cutscene)

## Project Restoration Implementation

- Fast Elegy Statues
- Fast Zora Swimming (fast swim without magic like N64)
- Ocarina Diving
- D-pad Transformation Masks
- D-pad Ocarina
- D-pad Arrow Switching
- Twinmold Health Fixes
- Deku movement speed fixes
- Giants Mask movement speed fixes
